class zensols.deeplearn.result.pred.PredictionsDataFrameFactory(epoch_result, stash, column_names=None, data_point_transform=None, batch_limit=9223372036854775807)[source]¶

Bases: object

Create a Pandas data frame containing results from a result as output from a ModelExecutor. The data frame contains the feature IDs, labels, predictions mapped back to their original value from the feature data item.

Parameters

  • epoch_result (EpochResult) – the test instance of an epoch result to populate the dataframe

  • stash (BatchStash) – the batch stash used in the ModelExecutor that was used to generate the results (not the DatasetSplitStash)

  • column_names (Optional[List[str]]) – the list of string column names for each data item the list returned from data_point_transform to be added to the results for each label/prediction

  • data_point_transform (Optional[Callable[[DataPoint], tuple]]) – a function that returns a tuple, each with an element respective of column_names to be added to the results for each label/prediction; if None (the default), str used (see the Iris Jupyter Notebook example)

  • batch_limit (int) – the max number of batche of results to output
